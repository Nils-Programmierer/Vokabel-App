# Vokabel-App
Die Vokabel-App ermöglicht es, Vokabeln aus Bildern zu scannen und bietet Abfragemethoden wie Karteikarten, Paare und Tests. Zudem gibt es Zugriff auf vordefinierte Vokabellisten für Englisch, um das Lernen zu erleichtern.
## Technologien
- HTML
- CSS
- JavaScript  
- [Material Design Lite](https://cdnjs.cloudflare.com/ajax/libs/material-design-lite/1.3.0/material.indigo-pink.min.css)
- [SweetAlert2](https://sweetalert2.github.io/)
- [Tesseract.js (OCR)](https://github.com/naptha/tesseract.js)
