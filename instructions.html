Anleitung

Vokabellisten:

Erstellen Sie ganz einfach eine neue Vokabelliste indem Sie einen Titel für die Liste eingeben. Vokabeln können Sie hinzufügen, wenn Sie eine gewünschte Liste auswählen und einen deutschen Text und einen englischen Text für die Vokabel eingeben. Alternativ können Sie auch eine bereits fertig erstellte Vokabelliste von uns zu einen bestimmten Thema auswählen, diese können Sie natürlich auch bearbeiten, neue Vokabeln hinzufügen und vieles mehr. Sie finden eine Übersicht aller Vokabellisten, wo Sie diese löschen oder durch Anklicken bearbeiten können. Wenn Sie eine Vokabelliste öffnen, werden alle Vokabeln angezeigt, und Sie können diese nach Bedarf löschen oder korrigieren oder den Titel der Vokabelliste ändern.


Vokabel-Karteikarten:

Dieser Modus eignet sich perfekt, um herauszufinden, ob Sie bereits alle Vokabeln kennen. Sie können auswählen, welche Vokabelliste und welche Sprache abgefragt werden sollen. Anschließend werden Ihnen die Vokabeln präsentiert. Drücken Sie auf das Fragezeichen, wenn Sie die Übersetzung der Vokabel nicht wissen, oder auf den Haken, wenn Sie es wissen. Wenn Sie auf den Button in der Mitte drücken wird die passende Übersetzung angezeigt. Am Ende erhalten Sie eine Zusammenfassung darüber, welche Vokabeln Sie wussten und welche nicht.


Vokabel-Paare:

In diesem Modus werden Sie feststellen, wie schnell Sie die Vokabeln zuordnen können. Sie können auswählen, welche Vokabelliste abgefragt werden soll. Anschließend werden Ihnen die Vokabeln angezeigt. Drücken Sie die Vokabeln an, die die gleiche Bedeutung haben. Sobald Sie alle Paare gefunden haben, wird Ihnen angezeigt, wie lange Sie gebraucht haben und welche Paare falsch waren.


Vokabel-Test:

Hier haben Sie die ideale Gelegenheit, Ihre Rechtschreibung bei den Vokabeln zu üben. Wählen Sie aus, welche Vokabelliste und welche Sprache abgefragt werden sollen. Anschließend wird Ihnen eine Vokabel angezeigt, und Sie sollen diese in der anderen Sprache eingeben. Ihnen wird dann angezeigt, ob Sie diese korrekt geschrieben haben. Am Ende erhalten Sie eine Übersicht, in der Sie sehen können, welche Wörter Sie falsch geschrieben haben und was Ihre Eingabe war.
